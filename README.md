
#  Budget Tracker Web App

A full-featured **Budget Tracker Web Application** that allows users to manage their daily expenses with real-time updates, authentication, data visualization, and CSV export.

Built as a **real-world practice project** to demonstrate full-stack web development skills using Firebase.
